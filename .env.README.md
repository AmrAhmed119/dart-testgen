# Environment Variables Setup

This project uses environment variables for configuration, particularly for the Gemini API key.

## Setup Instructions

1. **Copy the template file:**

   ```bash
   cp .env.dev .env
   ```

2. **Edit `.env` and add your actual API key:**

   ```
   GEMINI_API_KEY=your_actual_api_key_here
   ```

3. **Never commit `.env` to version control** - it contains sensitive information and is already in `.gitignore`

## Files

- **`.env`** - Your private environment variables (git-ignored)
- **`.env.dev`** - Public template file (safe to commit)

## Getting a Gemini API Key

Visit [Google AI Studio](https://aistudio.google.com/) to get a free API key.
